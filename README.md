🃏 Card Master Shop Website Project

โปรเจกต์จำลองเว็บไซต์สำหรับธุรกิจ **"ร้านขายการ์ดสะสม (Trading Card Shop)"** พัฒนาขึ้นด้วย HTML5 เพื่อแสดงโครงสร้างพื้นฐานของเว็บไซต์ธุรกิจ โดยเน้นการจัดลำดับเนื้อหาและการเชื่อมโยงลิ้งก์ระหว่างหน้าต่างๆ อย่างถูกต้อง

## 📂 โครงสร้างไฟล์ในโปรเจกต์

โปรเจกต์นี้ประกอบด้วยไฟล์ HTML จำนวน 4 ไฟล์หลัก ดังนี้:

### 1. `index.html` (หน้าแรก)
เปรียบเสมือนหน้าร้านออนไลน์ เป็นจุดเริ่มต้นของเว็บไซต์
- **Header:** แสดงชื่อร้านและเมนูนำทาง
- **Hero Section:** ส่วนต้อนรับขนาดใหญ่พร้อมภาพประกอบ
- **Featured Services:** แนะนำ 3 บริการหลักของร้านแบบย่อ

### 2. `about.html` (เกี่ยวกับเรา)
หน้าแสดงข้อมูลองค์กรและความน่าเชื่อถือ
- **Our Story:** ประวัติความเป็นมาของร้าน
- **Vision & Mission:** วิสัยทัศน์และพันธกิจ
- **Team:** แนะนำทีมงาน ผู้ก่อตั้ง และผู้เชี่ยวชาญในร้าน

### 3. `services.html` (บริการของเรา)
หน้าแสดงรายละเอียดสินค้าและบริการ
- **Service Details:** ข้อมูลเจาะลึกของบริการแต่ละประเภท (ส่งเกรด, ฝากขาย, ห้องแข่ง)
- **Price Table:** ตารางเปรียบเทียบแพ็กเกจสมาชิก (Comparison Table)

### 4. `contact.html` (ติดต่อเรา)
ช่องทางให้ลูกค้าติดต่อสอบถาม
- **Contact Form:** แบบฟอร์มสำหรับกรอกชื่อ อีเมล และข้อความ (HTML Form)
- **Address & Info:** ที่อยู่ เบอร์โทร และเวลาทำการ
- **Google Maps:** แผนที่ร้านแบบฝัง (Embed Map)

---

## 🛠 เทคโนโลยีที่ใช้
* **HTML5:** ใช้สำหรับวางโครงสร้างเนื้อหา (Structure only)
* **No CSS / No JS:** เวอร์ชันนี้เป็น Pure HTML เพื่อเน้นความเข้าใจใน Tag พื้นฐานและการเชื่อมโยงหน้า
